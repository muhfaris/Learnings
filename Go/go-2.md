## Workspace
Pada Go Workspace memiliki 3 folder
- src _ berisi tentang source code dari go_
- bin _ file binary hasil dari build _
- pkg _ paket yang dibutuhhkan dari aplikasi _


## PATH dari GO

``` 
 Workspace
 ---- [version go]
    --- src
        ---- github.com
            --- [username github] 
                --- [name repo] _ nama repository_
                    --- [name package] 
                        --- [aplikasi]
    --- pkg
        --- linux_amd64
         --- github.com
    --- bin
        --- hello-go
```
