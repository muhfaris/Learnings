# Run GO
## Membuat paket
- mkdir $GOPATH/src/github.com/user/hello

## Membuat file
- > cd $GOPATH/src/github.com/user/hello
- > touch hello.go

## Build aplikasi
- > go install github.com/user/hello

## Install aplikasi
- > cd $GOPATH/src/github.com/user/hello
- > go install

## Run Applikasi
- > $GOPATH/bin/hello
